# Module 02 | Practice

Practice assignments (PC 11–PC 20) and their solutions. This folder is the Flutter app.

## Layout

- **`lib/core/`** – Common code (theme, navigation).
- **`lib/features/home/`** – Home screen listing all assignments and navigating to each.
- **`lib/features/pc11_counter/`** … **`lib/features/pc20_image_rotation/`** – Each folder holds **both** the assignment (README + assets) **and** the solution code (presentation, and domain when used).
- **`Module_02_notes.md`** – Notes for this module.

## Run the app

From this folder (Practice):

```bash
flutter pub get
flutter run
```
