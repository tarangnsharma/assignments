# Module 04 | Practice

Practice assignments (PC 31–PC 43) and their solutions. This folder is the Flutter app.

## Layout

- **`lib/core/`** – Common code (theme, navigation).
- **`lib/features/home/`** – Home screen listing all practice assignments.
- **`lib/features/pc31_form_provider/`** … **`lib/features/pc43_firebase_auth/`** – Each folder holds **both** the assignment (README + assets when present) **and** the solution code (presentation, domain when used).
- **`Module_04_notes.md`** – Notes for the practice assignments.

## Run the app

From this folder (Practice):

```bash
flutter pub get
flutter run
```
