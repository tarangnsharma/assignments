## Per-assignment detail

- **GC 01** – Set `MaterialApp.title` in main.dart; for web this is the browser tab title.
- **GC 02** – Sum list of numbers (e.g. `fold`); optional Flutter screen to display result.
- **GC 03** – Generate Fibonacci (1, 1, 2, 3, 5, …) up to given max; optional Flutter screen.
- **GC 04** – Single screen: image (or placeholder), title; AppBar and body with contrasting colors.
- **GC 05** – Split sentence by whitespace, return word with max length; optional Flutter screen.
